# Librería Java - Trabajo Práctico Integrador
Trabajo práctico integrador de la materia Algoritmos 1 / 2025, como parte del programa en la carrera de **Licenciatura** en **Ciencia de Datos** de la **Universidad Nacional de San Martin(UNSAM)**.


# Título 
**Librería para gestión y procesamiento de datos tabulares**

# Descripción 
El trabajo práctico integrador consistió en el desarrollo de una librería utilizando lenguaje Java, destinado a gestionar, manipular y analizar datos en formato tabular(dos dimensiones).
Las principales funcionalidades consisten en carga, exportación, selección, ordenamiento, filtrado y agregación de datos.

El objetivo de este trabajo fue construir un proyecto aplicando el paradigma de **Programación Orientada a Objetos(POO)**,principios **SOLID**, modularización y patrones de diseño establecidos.

# Contenidos
* **Código_Java**: Código fuente del proyecto.
* **Diagrama de clases de la librería**: representación visual estática de módulos y relaciones entre clases.
* **Documentación del Trabajo Práctico**: objetivo del proyecto, requerimientos funcionales y no funcionales, desarrollo de módulos.
* **Documentación técnica de la librería**: Detalles de implementacion de modulos, métodos y ejemplos de uso.
* **Presentación TP librería Java**: presentación del proyecto finalizado.

# Funcionalidades principales
* Carga y descarga de datos en archivos csv.
* Creación y modificación de estructuras tabulares.
* Acceso a través de filas, columnas y celdas.
* Selección de datos.
* Filtro de datos.
* Copia y concatenación de estructuras.
* Ordenamiento condicional de datos.
* Muestreo aleatorio.
* Manejo de valores faltantes(NA).
* Visualización de tablas en formato texto.
